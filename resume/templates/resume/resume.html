3DSign CSCI 490 Capstone - CSU, Chico

January 2024 - May 2024

-

I designed software that allows a user to input a password on their PC using hand movements.

-

The hand movements are captured by a Raspberry Pi and an MPU9250 Inertial Measurement Unit connected via GPIO. The
movement data is interpreted onboard the Pi and then sent via Bluetooth to a receiving device. Password data is stored has
salted hash strings in a local shadow file.

-

This project uses C, GNU Make, petzval’s “btferret” Bluetooth library, I2C Bus, and Glibc’s Crypt library.
